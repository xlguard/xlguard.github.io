    在windows上部署基于hugo框架的博客

    第一步：下载hugo

    在https://github.com/gohugoio/hugo/releases 地址里面找到Windows的下载方式下载，这里本机下载的是64位的。

    并安装，安装的步骤很简单，直接解压就可以了。

    这里我们需要将hugo配置环境变量，在使用hugo version命令查看我们的hugo框架是否能运行。


    第二步：创建博客

    用管理员身份打开控制台，用hugo new site myblog命令创建自己的博客。

    我们能够通过命令cd myblog进入此文件夹时，则代表我们已经创建成功。

    第三步：下载主题，并应用

    在https://themes.gohugo.io/ 网站里面下载自己喜欢的主题

    下载完成之后，我们可以查看theme文件里面是否已经存在

    第四步：在本地启动博客，并创建文章

    在命令行中敲上 hugo server -t m10c –buildDrafts 命令

    在本地浏览器中查看

    在命令行敲击 hugo new post/blog.md 命令创建我们的第一篇文章

    我们创建的文章是在myblog目录下的content文件下的post目录下

    再次在本地启动博客服务

    第五步：在部署到GitHub上

    我们在GitHub中创建仓库

    把我们本地的项目和GitHub绑定起来，输入命令：hugo –theme=m10c –baseUrl=”你的GitHub地址” –buildDrafts

    完成之后会创建一个public文件，进入public文件夹下敲击 git commit -m “上传名称” 将本地项目上传


    敲击git remote add origin http://github.com/xlguard/xlguard.github.io.git 将本地的public和GitHub相关联

    关联之后我们启动该服务，敲击 git push -u origin master 命令，可能会让你输入账号和密码，输入就行

    刷新GitHub地址并打开里面内容
